🔢 Place Value

Every digit in a number has a place and a value.

TEN-THOUSANDSTHOUSANDSHUNDREDSTENSONES
47305
40,0007,00030005

💡 The number is:

47,305 = 40,000 + 7,000 + 300 + 0 + 5

🎭 Face Value vs Place Value

Face Value

The digit itself. In 47,305 — face value of 7 is 7

Place Value

Digit × its position. In 47,305 — place value of 7 is 7,000

⚠️ Common Mistake

Students confuse face value and place value. Remember: Face value never changes! 7 is always 7.

📏 Comparing Numbers

To compare two numbers:

Step 1:

Count the digits. More digits = bigger number

Step 2 (if equal digits):

Compare digit by digit from LEFT to RIGHT

Compare: 4,567 vs 4,589
Both have 4 digits. Compare: 4=4, 5=5, 6<8 → 4,567 < 4,589

🌟 Factors & Multiples

Factors of 12:

Numbers that divide 12 exactly: 1, 2, 3, 4, 6, 12

⚡ Trick:

Always start from 1. Find pairs: 1×12, 2×6, 3×4

Multiples of 4:

4, 8, 12, 16, 20, 24... (keep adding 4)

⚠️ Remember:

1 is a factor of EVERY number. Every number is a multiple of itself!

⚡

Greatest Number Trick

To form the GREATEST number:

Arrange all digits in DESCENDING order (largest first)

Digits: 3, 7, 1, 9, 4
Greatest: 97,431
🔻

Smallest Number Trick

To form the SMALLEST number:

Arrange all digits in ASCENDING order (smallest first)

Digits: 3, 7, 1, 9, 4
Smallest: 13,479

⚠️ If 0 is present:

0 cannot be first digit! Put the next smallest digit first.

➕ Addition Properties

Commutative

a + b = b + a → 23 + 45 = 45 + 23 = 68

Associative

(a+b)+c = a+(b+c) → (12+8)+5 = 12+(8+5) = 25

Identity

a + 0 = a → any number + 0 = same number

✖️ Multiplication Table Tips

× 9 Finger Trick:

Hold up 10 fingers. Fold the Nth finger for 9×N. Left side = tens, right = ones!

× 11 Trick (2-digit):

47 × 11 = 4_(4+7)_7 = 4_11_7 → carry the 1 → 517

Doubling & Halving:

24 × 5 = 24 × 10 ÷ 2 = 240 ÷ 2 = 120

➗ Division Concept

Dividend ÷ Divisor = Quotient

42 ÷ 6 = 7

Remainder Check:

Dividend = Divisor × Quotient + Remainder

Relationship:

Division is reverse of multiplication
6 × 7 = 42 → 42 ÷ 6 = 7

🎭 Real-Life Scenario

🍫 Rahul has 3 boxes with 48 chocolates each. He gives away 27 chocolates. How many remain?

Step 1: Find total

3 × 48 = 144 chocolates

Step 2: Subtract

144 − 27 = 117 chocolates remain

⚡ Shortcut: 144 − 27

144 − 30 + 3 = 114 + 3 = 117 ✓

⚡ Multiplication Tricks

Multiply by 25:

Divide by 4, multiply by 100
48 × 25 = 48÷4 × 100 = 12 × 100 = 1200

Multiply by 99:

Multiply by 100, subtract the number
37 × 99 = 3700 − 37 = 3663

Square of numbers ending 5:

n5² = n(n+1) then append 25
35² = 3×4=12, append 25 → 1225

⚡ Addition/Subtraction Tricks

Add near-round numbers:

236 + 99 = 236 + 100 − 1 = 335

Subtract near-round numbers:

500 − 197 = 500 − 200 + 3 = 303

BODMAS Rule:

Brackets → Of → Division → Multiplication → Addition → Subtraction

🍕 What is a Fraction?

A fraction represents a part of a whole.

1/2 of a pizza:
1/2
1/2
3/4 of a bar:
3/4

Numerator / Denominator

3/4 → 3 = numerator (parts taken), 4 = denominator (total parts)

🏷️ Types of Fractions

Proper Fraction:

Numerator < Denominator → 3/5, 2/7

Improper Fraction:

Numerator ≥ Denominator → 7/3, 5/5

Mixed Number:

Whole + fraction → 2¾ = 2 + 3/4

Equivalent Fractions:

1/2 = 2/4 = 3/6 = 4/8 (multiply top & bottom by same number)

🔄 Comparing Fractions

Same Denominator:

3/7 vs 5/7 → Compare numerators → 5/7 > 3/7

Same Numerator:

3/4 vs 3/7 → Smaller denominator = bigger fraction → 3/4 > 3/7

Cross Multiply:

2/3 vs 3/5 → 2×5=10 vs 3×3=9 → 10>9 → 2/3 > 3/5

⚡ Fraction Shortcuts

Convert Mixed to Improper:

2¾ = (2×4+3)/4 = 11/4
Formula: (Whole×Denominator+Numerator)/Denominator

Adding Same Denominator:

2/7 + 3/7 = (2+3)/7 = 5/7

Simplify Fractions:

Find HCF of top & bottom, then divide both
12/18 → HCF=6 → 2/3

🍰 Fraction Word Problem Trick

🎂 A cake is cut into 8 equal pieces. Sara eats 3 pieces. What fraction is eaten?

Answer: 3/8

3 pieces eaten out of 8 total = 3/8

Fraction remaining:

8/8 − 3/8 = 5/8

⏱️ Time Calculation Tricks

Finding Duration:

End time − Start time

AM/PM Rule:

12 AM = midnight (start of day)
12 PM = noon (middle of day)

24-hour conversion:

Add 12 to PM times (1 PM = 13:00)
12 AM = 00:00, 12 PM = 12:00

💰 Money Tricks

Change Calculation:

Change = Amount given − Cost
Given ₹100, cost ₹67 → ₹100−₹67 = ₹33 change

Best Buy Trick:

Compare cost per unit: Divide total price by quantity. Smaller = better deal!

📏 Perimeter Formulas

Square:

P = 4 × side

Rectangle:

P = 2 × (length + breadth)

Triangle:

P = sum of all 3 sides

⚠️ Area vs Perimeter:

Perimeter = boundary (fence)
Area = surface (carpet)

📦 Area Formulas

Square:

A = side × side = side²

Rectangle:

A = length × breadth

Triangle:

A = ½ × base × height

🔄 Symmetry

Line of Symmetry:

A line that divides a shape into two EQUAL mirror halves

Lines of symmetry:

Square: 4 lines
Rectangle: 2 lines
Circle: Infinite lines
Equilateral triangle: 3 lines

⚡ Geometry Tricks

Perimeter to Area (Square):

If P = 20 cm → side = 20÷4 = 5 cm → Area = 5² = 25 cm²

Area to Perimeter (Square):

If A = 64 cm² → side = √64 = 8 cm → P = 4×8 = 32 cm

Rectangle Trick:

If P = 26, L = 8 → B = (26÷2)−8 = 13−8 = 5

📐 Angle Tricks

Types of Angles:

Acute: <90° | Right: =90° | Obtuse: 90°–180°
Straight: =180° | Reflex: >180°

Clock Angle Trick:

Each hour mark = 30° (360°÷12)
Hour hand moves 0.5° per minute!

🔤 Coding-Decoding

Letter shift trick:

If A=1, B=2...Z=26, decode by finding letter positions

If CAT = 3-1-20, then DOG = ?

D=4, O=15, G=7 → DOG = 4-15-7

🔍 Odd One Out

Strategy:

Find the common property of 3 items. The 4th that doesn't fit = Odd one out

Apple, Mango, Banana, Carrot

Carrot is a vegetable; others are fruits

2, 4, 6, 9, 8

9 is odd; others are even

🔢 Number Analogies

Method:

Find the relationship between the first pair, apply to second pair

3 : 9 :: 5 : ?

3 × 3 = 9, so 5 × 5 = 25

2 : 8 :: 3 : ?

2³ = 8, so 3³ = 27

🔄 Mirror Images

Rule:

Left-Right gets flipped in a mirror image. Up-Down remains same.

For Letters:

Draw axis, flip left ↔ right
b ↔ d, p ↔ q, A stays A (symmetric!)

🧭 Direction Sense

Compass Rules:

Face North → Turn Right = East → Turn Right again = South → Turn Right = West

🔢 Pattern Finding

Number Series Tricks:

• Check differences: 2,5,8,11 → +3 each
• Check ratios: 2,4,8,16 → ×2 each
• Check squares: 1,4,9,16,25 → 1²,2²,3²,4²,5²

Find next: 3, 6, 12, 24, ?

Pattern: ×2 each → 48

⚡ Elimination Method

For MCQs:

1. Eliminate clearly wrong options first
2. Between 2 options, apply the rule carefully
3. Check your answer by substituting back
